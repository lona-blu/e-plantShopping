# e-plantShopping
Paradise Nursery – Shopping Cart Application

Paradise Nursery is a dynamic online plant shop web application built as a final project assignment. The app allows users to browse a variety of houseplants, view plant details, add items to a shopping cart, and manage quantities while seeing the total cost update in real time.

This project demonstrates core front-end development skills using React and Redux, focusing on building a functional and user-friendly e-commerce experience.

Features

Browse plants by category

View plant images, names, descriptions, and prices

Add plants to the shopping cart

Increase or decrease quantities

Remove items from the cart

Dynamic cart item count in the navbar

Automatic total price calculation

Disable “Add to Cart” button once a product is added

Navigation bar with Home, Plants, and Cart links

Landing page with background image and “Get Started” button

Technologies Used

React.js

Redux Toolkit

React Router

JavaScript (ES6+)

HTML5

CSS3
